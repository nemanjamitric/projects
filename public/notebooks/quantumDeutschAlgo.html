Deutschov algoritem v Qiskitu¶

Implementacija Deutschov algoritem z uporabo kvantnega simulatorja v Qiskit. Cilj algoritma je ugotoviti, ali je neka funkcija konstantna ali uravnotežena, z le eno samo kvantno poizvedbo.

IBM Quantum Circuit example

Priprava okolja¶

Najprej uvozimo potrebne knjižnice in ustvarimo registre ter osnovno circuit.

In [1]:
%matplotlib inline
from qiskit import QuantumRegister, ClassicalRegister, QuantumCircuit
from qiskit_aer import AerSimulator
from qiskit.visualization import plot_histogram
from qiskit import transpile
from matplotlib import pyplot as plt

def circuitSetup():
    qreg = QuantumRegister(2, 'q')
    creg = ClassicalRegister(1, 'c')
    circuit = QuantumCircuit(qreg, creg)
    return circuit, qreg, creg

def padCounts(counts):
    return {'0': counts.get('0', 0), '1': counts.get('1', 0)}

Deutschov algoritem z izbrano orakelj funkcijo¶

Ta funkcija zgradi vezje za Deutschov algoritem glede na podano funkcijo f(x). Možni oraklji so:

  • "konstanta" za f(x) = 0
  • "uravnotezena" za f(x) = 1
In [2]:
def deutschAlgorithm(oracleType):
    circuit, qreg, creg = circuitSetup()
    
    circuit.x(qreg[1])
    circuit.id(qreg[0])
    circuit.h(qreg[0])
    circuit.h(qreg[1])

    circuit.barrier()
    
    if oracleType == "konstanta":
        circuit.x(qreg[1])
    elif oracleType == "uravnotezena":
        # circuit.x(qreg[0])
        circuit.cx(qreg[0], qreg[1])
        # circuit.x(qreg[0])
    
    circuit.barrier()
    
    circuit.h(qreg[0])
    circuit.measure(qreg[0], creg[0])

    # @columns [0,1,1,2,2,3,3,4,4,5,6]
    
    return circuit

Primer: Konstantna funkcija¶

Spodaj je primer, kjer je orakelj konstantna funkcija f(x) = 0. Rezultat mora biti vedno 0.

In [3]:
circuitConstant = deutschAlgorithm("konstanta")
circuitConstant.draw('mpl')

simulator = AerSimulator()
circuitConstant = transpile(circuitConstant, simulator)
resultConstant = simulator.run(circuitConstant, shots=1024).result()
countConstant = resultConstant.get_counts()
plot_histogram(padCounts(countConstant))
plt.show()

Primer: Uravnotežena funkcija¶

Zdaj uporabimo orakelj, ki predstavlja uravnoteženo funkcijo f(x) = x. Rezultat mora biti 1.
